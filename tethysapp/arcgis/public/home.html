The Volume of the Big Butte

By: Hani Abu-Hmeidan and William Shelton

This is a project for our Geospatial Software Development class (CEen 514) at Brigham Young University. For this project we created a Tethys App that allows the user to import an Elevation to compute the volume of material to be removed and display a polygon representing the area of interest.

Our task:

1. Create a Geoprocessing tool that calculates the Volume of the Big Butte.

2. Link the tool to the BYU Geoserver.

3. Excute the tool and display results on the Map.

How to use the map:

1. Insert an Elevation in the textbox. The MINIMUM elevation is 1666 feet.

2. Click on "RUN".

3. Wait for the tool to run and the volume will be displayed.
